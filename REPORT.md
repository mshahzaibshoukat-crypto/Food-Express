# Assignment 4 Learning Report

## Project Overview
For this assignment, I upgraded the static "Food Express" website into a dynamic full-stack web application. The frontend is built with HTML, CSS (Tailwind), and Vanilla JavaScript, while the backend is powered by Node.js and Express.

## Features Implemented
- **Node.js & Express Backend**: Set up a robust server to handle RESTful API requests.
- **REST APIs**: Developed routes for getting menu items, interacting with the shopping cart, and submitting contact forms.
- **CRUD Functionality**: The cart API allows users to Create (add items), Read (view items), Update (change quantities), and Delete (remove items).
- **In-Memory Database**: For simplicity and ease of deployment, data is stored in memory using JavaScript arrays on the server.
- **Frontend Integration**: Updated static HTML pages to fetch data asynchronously from the backend APIs using the native JavaScript `fetch()` API.

## Learning Resources Used
In completing this assignment, the following resources were consulted and utilized:
1. **Node.js Documentation**: For understanding the basics of creating a server and managing the environment.
2. **Express.js Documentation**: To learn how to set up routing, middleware, and API endpoints efficiently.
3. **MDN Web Docs (Mozilla Developer Network)**: For comprehensive guides on using the `fetch` API to make HTTP requests from the frontend to the backend.
4. **Stack Overflow / Community Forums**: For troubleshooting minor bugs related to CORS and resolving JSON parsing errors during development.
5. **Tailwind CSS Documentation**: To maintain and adjust the responsive styling of the newly integrated dynamic elements.
